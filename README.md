# CUET Scores Data Visualisation, Analysis and Representation

## 📊 Project Overview

This project is a comprehensive **data visualization, analysis, and representation of CUET score-related information** using Python and Matplotlib. It presents multiple graphical insights in a clear 2x2 subplot layout for easy interpretation and comparison.

## 🎯 Objectives

- Represent, analyze, and visualize data from Pandas Series, DataFrames, NumPy Arrays, and Sets format
- Uncover meaningful insights from CUET score data
- Present department-wise cutoff scores and student distributions
- Compare year-over-year score distributions
- Analyze student satisfaction across branches

## 📈 Visualizations Included

### 1. **Box Plot** - CUET Cutoff Scores for Departments
   - Shows score distribution for each department
   - Displays median, quartiles, and outliers
   - Includes mean values (green dots)

### 2. **Pie Chart** - Student Strength in Each Department
   - Represents student enrollment percentages by department
   - Highlights CS (highest) and Botany (lowest) departments
   - Shows department-wise distribution

### 3. **Histogram** - CUET Scores Distribution Comparison
   - Compares Previous Year vs. Current Year scores
   - Identifies trends and improvements
   - Visualizes score frequency distribution

### 4. **Bar Chart** - Student Satisfaction with Admitted Branches
   - Measures satisfaction scores (0-10 scale) for each branch
   - Helps assess student contentment across departments

## 📋 Features

- ✅ Multiple chart types (Boxplot, Pie Chart, Histogram, Bar Chart)
- ✅ Color-coded visualizations for clarity
- ✅ 2x2 subplot layout for side-by-side comparison
- ✅ Data-driven insights visualization
- ✅ Professional formatting and grid support

## 🛠️ Technologies Used

- **Python 3.x**
- **Matplotlib** - Data visualization
- **NumPy** - Numerical computing
- **Pandas** - Data manipulation

## 📊 Data Structure

The project works with:
- **Dictionary data** for department cutoff scores
- **Lists** for student counts and satisfaction ratings
- **NumPy arrays** for historical score comparisons
- **Matplotlib subplots** for organized visualization

## 📝 Key Insights

- CS Department has the highest cutoff scores (320-350)
- Botany Department has the lowest student enrollment (~9%)
- Current year CUET scores show higher performance compared to previous year
- Student satisfaction varies across branches (6.7-9.2 scale)
